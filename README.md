## Tech stack
- Kotlin based, Coroutines for asynchronous
- Dagger for dependency injection
- Jetpack:
  - Lifecycle
  - ViewModel
  - LiveData
- Architecture:
  - MVVM Architecture
- Navigation Component
- Retrofit2 & OkHttp3
- Gson
- Picasso

## Open Api:

PureWeather client using the [Weatherbit.io](https://www.weatherbit.io/features). Its a RESTful API.
